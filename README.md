# Quint Landing Page

Static landing page for Quint Security.

## Deploy to Vercel

```bash
npm i -g vercel
cd quint-landing
vercel
```

Or connect the repo to Vercel dashboard for auto-deploys.

## Structure

- `index.html` — single-page static site
- `vercel.json` — Vercel routing config
